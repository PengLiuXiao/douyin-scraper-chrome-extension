# 🎬 Douyin URL 抓取插件 (Douyin URL Extractor)

专为抖音网页端设计的高性能、免配置数据采集 Chrome 浏览器扩展。通过 **API 数据拦截** 与 **DOM 智能降级** 双通道抓取技术，精准、高速地提取视频多维度元数据。

无需编写 Python/Selenium 脚本，无需配置复杂的代理和 Cookie，浏览器原生运行。支持**一键自动滚动爬取、批量关键词自动化搜索采集、直存本地目录以及一键导出 CSV/XLSX 报表**，为短视频数据分析、运营选品及学术研究提供强有力的数据支持。

---

## ✨ 核心特性

*   🎯 **API 拦截，数据零失真**：直接拦截抖音网页端 API 响应，捕获无损的原始 JSON 数据。相比传统 DOM 渲染抓取，速度更快、数据更完整、准确率高达 100%。
*   🚀 **零配置，即开即用**：无需任何账号或 Token 配置，直接打开抖音页面点击扩展即可立即读取数据。
*   📊 **12 个核心维度数据**：提取包括视频 ID、视频链接、标题、作者昵称、作者 ID (SecUID)、视频时长、点赞数、评论数、分享数、播放量、发布时间、视频封面链接在内的完整字段。
*   🔄 **智能滚动与自动停止**：一键开启自动循环滚动，支持智能停止机制（无新数据自动超时停止、连续滚动未更新自动停止），防止资源浪费与风控限制。
*   📥 **批量关键词搜索**：支持导入 `.csv` 或 `.xlsx` 关键词列表，全自动执行批量搜索和抓取任务，支持任务队列顺序执行。
*   📂 **直存本地目录**：支持通过 Web File System Access API 绑定本地输出目录，批量爬取时自动将每个词的数据独立保存为本地文件，断电/异常中断也不丢数据。
*   🗂️ **多标签页独立运行**：每个 Chrome 标签页独立运行，拥有各自独立的缓存数据、采集任务和配置选项，支持多开并行采集。
*   🪟 **Side Panel 侧边栏持久化**：支持将扩展固定在 Chrome 侧边栏（Side Panel）中展示。在抖音页面切换时，侧边栏保持打开且不中断，极大提升使用体验。
*   💾 **多格式一键导出**：内置 Handsontable 高性能电子表格，支持在界面中直接预览、筛选、编辑数据，支持一键导出 CSV、Excel (XLSX) 或复制数据至剪贴板。

## 🛠️ 安装方法

由于本扩展为定制的高级数据采集工具，请通过以下步骤在 Chrome 浏览器中手动加载使用：

1.  **下载源码**：将本仓库的代码下载并解压到您的本地电脑上。
2.  **打开扩展程序页面**：在 Chrome 浏览器地址栏中输入 `chrome://extensions/` 并回车。
3.  **启用开发者模式**：在扩展程序页面右上角，将 **「开发者模式」** 开关切换为 **开启** 状态。
4.  **加载已解压的扩展程序**：点击左上角的 **「加载已解压的扩展程序」** 按钮。
5.  **选择项目文件夹**：在弹出的文件选择器中，选择解压出的项目根文件夹（即包含 `manifest.json` 的目录）。
6.  **固定扩展**：加载成功后，点击浏览器右上角的拼图图标（扩展程序），找到 **「DouyinURL提取Pro」**，点击图钉图标将其固定在工具栏。

## 📖 使用指南

### 模式一：单页面常规数据采集
1.  打开浏览器并登录抖音网页版，访问您需要采集的页面（如：某个博主的主页、热搜榜、话题页等）。
2.  点击浏览器工具栏的 **DouyinURL提取Pro** 图标，打开侧边栏或弹窗，当前页面已拦截的视频数据列表会立即呈现。
3.  点击 **「开始爬取」**，扩展将开始自动向下滚动页面并抓取新出现的数据。
4.  采集达到预期数量后，点击 **「停止爬取」**。
5.  在右侧控制区点击 **「CSV」**、**「XLSX」** 或 **「复制全部」** 导出当前采集到的完整数据。

### 模式二：单关键词快速提取
1.  在抖音任意页面打开扩展面板。
2.  在顶部的 **「抖音关键词视频提取」** 输入框中，输入要搜索的词（例如：`智能AI`）。
3.  点击 **「搜索并提取」** 按钮，浏览器会自动跳转到抖音对应的搜索结果页面，并立即启动自动滚动采集。

### 模式三：批量关键词全自动爬取
如果您有大量的关键词需要依次搜索采集，可以使用此功能：
1.  **准备关键词文件**：新建一个 Excel (XLSX) 或 CSV 文件，其中一列的表头命名为 `关键词`（或 `keyword`、`keywords`，若无表头则默认取第一列）。在此列下方填入您要批量搜索的词。
2.  **导入文件**：在扩展面板中，点击 **「批量导入关键词」** 并选择刚才准备好的文件。
3.  **绑定本地保存目录（强烈推荐）**：
    *   导入后系统会提示您选择一个本地保存目录。
    *   点击 **确认** 并选择本地电脑上的任意文件夹（并授予浏览器写入权限）。
    *   绑定后，**每完成一个关键词的抓取，扩展会自动将该词的数据直接保存为该文件夹下的一个 CSV 文件**（命名格式如 `关键词_日期.csv`）。即使浏览器崩溃或网络断开，已完成的数据也已安全落盘。
    *   *如果点击取消，所有数据只会暂存在浏览器内存中，待全部任务完成后可手动合并导出。*
4.  **全自动运行**：点击开始后，扩展将全自动运行。它会自动跳转到对应关键词的搜索结果，按您的配置进行滚动采集，抓取完毕后自动保存，再自动切换到下一个词。您可以在 **「批量搜索进度」** 面板中实时查看任务进度。
5.  **下载合并数据**：批量任务全部运行结束后，您可以点击 **「下载合并CSV」**，一键下载将所有关键词采集到的数据整合在一起的汇总表格（新增 `搜索关键词` 列用作区分）。

## ⚙️ 采集配置说明

您可以在扩展面板中根据当前网络状况和数据需求微调以下参数：

| 配置项 | 默认值 | 作用与建议 |
| :--- | :---: | :--- |
| **最短延迟(≥0)** | `1.5` 秒 | 触发页面滚动的最短等待时间。数值越低抓取越快。**建议保持在 1.5s 或以上**，可有效降低被抖音系统限流或触发验证码的风险。 |
| **最长等待(>延迟)** | `20` 秒 | 等待网络或页面响应的超时时间。如果在此时间内页面无任何反应，将尝试再次触发滚动。 |
| **时间上限(5-60)** | `15` 秒 | **智能停止条件一**：如果在设定的秒数内（如 15 秒）未采集到任何**全新**的视频数据，扩展将认为已到达页面底部并自动停止爬取。 |
| **连续无新增(1-10)** | `3` 次 | **智能停止条件二**：页面连续触发滚动指定次数后，若采集到的视频总数没有任何增加，将自动停止爬取，防止在底部无限空刷。 |
| **下载子目录** | `douyin-url-extractor` | 当您直接点击 CSV / XLSX 按钮导出文件时，文件会默认保存至您系统下载目录（Downloads）下的该指定子文件夹内，方便文件的统一归档管理。 |

## 📊 提取字段详解

本扩展共为您提取以下 12 个维度的核心字段，结构清晰，便于后续进行数据分析：

1.  **视频ID** (`aweme_id`)：抖音平台视频的加密唯一标识码。
2.  **视频链接** (`url`)：该视频的直接播放与访问链接。
3.  **视频标题** (`desc`)：视频的文案、标题及包含的话题标签。
4.  **作者** (`nickname`)：发布该视频的创作者昵称。
5.  **作者ID** (`sec_uid`)：创作者的加密唯一 ID，用于通过链接直接跳转到其主页。
6.  **时长** (`duration`)：视频的播放总时长（例如：`15s`、`120s`）。
7.  **点赞数** (`digg_count`)：视频当前获得的点赞数量。
8.  **评论数** (`comment_count`)：视频的评论总数。
9.  **分享数** (`share_count`)：视频被分享转发的次数。
10. **播放量** (`play_count`)：视频的总播放观看次数。
11. **发布时间** (`create_time`)：视频发布的精确时间（已自动转化为可读的 `YYYY-MM-DD HH:mm:ss` 格式）。
12. **封面链接** (`cover`)：视频封面的高清图片直链。

## ⚠️ 使用注意事项

1.  **登录抖音账号**：使用前请确保您已在浏览器中登录了抖音账号。未登录账号访问抖音时，抖音会频繁弹出登录拦截框，并限制页面滚动和数据加载。
2.  **避免过快采集**：虽然本扩展支持较短的滚动延迟，但连续、高速地滚动可能会导致抖音系统暂时限制您的请求，或弹出拼图滑块验证。如遇此情况，请手动完成验证，或在配置中适当调大「最短延迟」。
3.  **浏览器权限授予**：
    *   首次使用「批量关键词导入」并选择本地目录时，浏览器会弹出权限申请，请点击 **「允许/保留更改」**，以允许扩展将数据自动写入您指定的文件夹。
    *   文件下载需要授予 `downloads` 权限，扩展已内置，通常无需手动配置。
