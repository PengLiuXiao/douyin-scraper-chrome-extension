# DouyinURL提取Pro

专为抖音平台设计的 Chrome 扩展，通过 API 拦截 + DOM 智能降级，提取视频数据并支持 CSV / XLSX 导出。

## 功能特性

- 🎯 **API 智能拦截** — 直接从抖音 API 响应捕获原始数据，准确可靠
- 🎨 **抖音专用** — 专门为抖音平台优化，支持搜索、主页、关注、合集、话题等页面
- 📊 **12 个数据字段** — 视频ID、链接、标题、作者、作者ID、时长、点赞数、评论数、分享数、播放量、发布时间、封面链接
- 💾 **多格式导出** — 支持 CSV、XLSX 格式导出及一键复制
- 🔄 **自动滚动采集** — 无限滚动模式，自动翻页抓取大量数据
- 🎛️ **实时预览编辑** — Handsontable 表格预览，支持列重命名与删除
- 📥 **批量导入搜索** — 支持 CSV/XLSX 批量导入关键词，多任务顺序执行、独立 CSV 自动保存及一键合并导出
- 🛑 **无增量自动停止** — 超时或连续滚动无新增数据时自动停止，避免无用开销
- 🪟 **侧边栏模式** — 支持 Chrome 侧边栏（Side Panel）固定展示，无需反复点击图标
- 🗂️ **多 Tab 独立状态** — 每个标签页独立维护采集状态，互不干扰
- 📁 **自定义下载目录** — 文件通过 `chrome.downloads` API 下载，支持配置自定义子目录（默认 `douyin-url-extractor`）

## 🛠️ 技术架构

### 数据提取机制

```
用户浏览抖音
    │
    ├─ 1️⃣ API 拦截（inject.js，MAIN world）← 主力通道
    │      拦截 fetch / XHR → 解析 JSON → postMessage → content.js
    │      覆盖 11 个 API 端点，获取完整结构化字段
    │
    └─ 2️⃣ DOM 降级（syncDOMToCache，ISOLATED world）← 备用通道
           策略一：优先 data-e2e 语义属性定位视频卡片
           策略二：降级到 a[href*="/video/"] 链接扫描
```

## 🚀 安装使用

### 安装方法

1. 下载项目源码
2. 打开 Chrome 浏览器
3. 访问 `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择项目文件夹

### 使用流程

1. **打开抖音** — 在浏览器中访问 https://www.douyin.com
2. **启动扩展** — 点击工具栏中的扩展图标
3. **自动初始化** — 扩展自动通过 API 拦截获取当前页面数据
4. **开始采集** — 点击「开始爬取」，扩展自动滚动并持续收集数据
5. **导出数据** — 点击 CSV 或 XLSX 按钮下载完整数据

### 关键词搜索（快捷方式）

在弹窗顶部「抖音关键词视频提取」区域输入关键词，点击「搜索并提取」，扩展将自动跳转到搜索页并开始采集。

### 批量关键词导入搜索

1. **准备关键词文件** — 准备一个 `.csv` 或 `.xlsx` 文件，其中包含一列名为 `关键词`、`keyword` 或 `keywords` 的列（或无表头，以第一列为准）。
2. **导入文件** — 点击弹窗中的「批量导入关键词」按钮选择您的文件。
3. **选择保存目录（可选）** — 在弹出的提示中可点击「确定」选择一个本地保存目录（借助 File System Access API），此后每个关键词采集完成后会自动将数据以 CSV 格式保存到该目录中；若选择「取消」，则各关键词结果将仅暂存在内存中，待全部完成后导出。
4. **自动运行与进度监控** — 扩展将自动遍历关键词，更新页面并启动滚动采集。您可以在弹窗下方的「批量搜索进度」面板实时查看全局以及当前关键词的进度，也可随时中途「取消批量」任务。
5. **下载合并数据** — 批量任务全部完成后，可点击「下载合并CSV」一键导出所有关键字的聚合数据（新增「搜索关键词」一列用作标识）。

## ⚙️ 配置选项

### 爬取配置

- **最短延迟（≥0）**：触发滚动的最短等待时间（秒，默认 **1.5 秒**）
- **最长等待（>延迟）**：等待页面响应的超时时间（秒，默认 20 秒）
- **时间上限（5-60）**：连续未采集到新数据的最长超时时间（秒，默认 15 秒）
- **连续无新增（1-10）**：连续触发滚动多少次仍无新数据即停止（次，默认 3 次）
- **下载子目录**：文件保存到浏览器下载目录下的指定子文件夹（默认 `douyin-url-extractor`）
- **预览限制**：弹窗内默认显示 1000 条记录，完整数据通过导出获取
