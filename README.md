# DouyinURL提取Pro

AI智能检测抖音视频数据，一键提取URL、aweme-id等15个字段，支持CSV导出

## 功能特性

- 🎯 **AI智能检测** - 基于Instant Data Scraper技术，自动识别页面表格数据
- 🎨 **抖音专用** - 专门为抖音平台优化，提取视频特有字段
- 📊 **15个数据字段** - 全面提取aweme-id、video-url、author-id等关键信息
- 💾 **多格式导出** - 支持CSV、XLSX格式导出
- 🔄 **批量处理** - 支持翻页爬取，一次性获取大量数据
- 🎛️ **可视化编辑** - 实时预览和编辑数据字段

## 🛠️ 技术架构

### 核心设计模式
完全参考Instant Data Scraper的成功模式：

1. **自动检测** → `findTables()` - AI分析页面结构
2. **高亮选择** → `nextTable()` - 用户可选择不同检测结果  
3. **标记滚动** → `getScrollButton()` - 检测滚动区域
4. **智能爬取** → `clickScroll()` - 自动滚动并抓取数据
5. **数据导出** → CSV/Excel格式导出

## 🚀 安装使用

### 安装方法
1. 下载项目源码
2. 打开Chrome浏览器
3. 访问 `chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择项目文件夹

### 使用流程
1. **访问抖音网站**：打开 https://www.douyin.com
2. **启动AI检测**：点击扩展图标，AI自动分析页面
3. **确认检测结果**：查看高亮的视频区域，可点击"尝试其他视频区域"
4. **设置滚动**：点击"定位向下滚动"（可选）
5. **开始爬取**：点击"开始智能爬取"进行多页数据抓取
6. **导出数据**：选择CSV或Excel格式下载

### 高级功能
- **表格预览**：实时预览抓取的数据
- **字段定制**：删除不需要的数据列
- **爬取配置**：调整滚动延迟和等待时间
- **批量操作**：一次处理大量视频数据
